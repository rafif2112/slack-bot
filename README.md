# Fidev Bot 🚀

A modern Slack Bot built to improve developer productivity with useful commands and future AI-powered assistance.

Fidev Bot provides a simple way for users to access helpful tools directly inside Slack. This project includes a custom landing page to showcase available commands and make the bot easier to understand.

## ✨ Features

### Slack Bot Commands

Fidev Bot supports multiple slash commands inside Slack:

* `/fidev-joke`

  * Generate a random joke

* `/fidev-catfact`

  * Get a random cat fact

* More commands coming soon...

### Command Showcase UI

A dedicated frontend interface that displays available commands with:

* Clean developer-focused design
* Command descriptions
* Interactive command preview
* Responsive layout

The goal is to make users understand what the bot can do without testing every command manually.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/fidev-bot.git

cd fidev-bot
```

---

## Frontend Setup

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## Backend Setup

Install dependencies:

```bash
npm install
```

Create environment file:

```env
SLACK_BOT_TOKEN=
SLACK_SIGNING_SECRET=
PORT=3000
```

Run server:

```bash
npm start
```
